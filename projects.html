Projects
Stage Ready

Python, React, Next.js, LangChain, Flask, Tailwind CSS, shadcn, Groq, hume

Stage Ready is a web app designed to help users practice public speaking in a judgment-free, AI-powered environment. By analyzing facial expressions and speech patterns, the app provides constructive feedback and simulates a Q&A session based on the user's presentation. Built with React, Flask, Groq, LangChain and Hume AI, it empowers users to refine their delivery without needing a live audience.

YGO Market Watch

Python, Django, Bootstrap, HTML/CSS, SQLite

Created a website utilizing the Django framework, integrated an external API allowing the user to query detailed Yu-Gi-Oh trading card information from an SQLite database, track price changes over time, and facilitate easy card price comparisons.

Stock Picker

Python, Flask, AWS, JavaScript/Jquery, AJAX, HTML/CSS

A web app that displays the latest information about a desired stock. Hosted on AWS, this app uses Flask for the backend to retrieve and send data to the TiingoStocks API using AJAX for a smooth user experience.
